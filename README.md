# chouwen-rs
**高性能 Rust 原生 | 悬浮窗常驻 | 大屏投影 | 名单导入 | 抽取统计 | 纯本地无广告**
## 📢 项目简介
**抽问** 是一款由 Rust 语言独立开发的课堂随机抽取/点名工具，专为教师场景设计。
实现**桌面悬浮窗**（不遮挡课件）、**大屏展示**（投影清晰）、**名单批量导入**、**抽取历史统计**四大核心刚需，
启动快、占用低、稳定不卡顿，全程本地运行，保护学生隐私。

## ✨ 核心功能
- ✅ 悬浮窗模式：置顶、半透明、屏幕边角吸附，最小化不遮挡教学
- ✅ 大屏展示：全屏高亮动画，适配教室投影/大屏
- ✅ 名单导入：支持通过json/直接输入学生姓名快速导入学生名单
- ✅ 抽取统计：自动记录抽中次数，支持查看与导出
- ✅ 纯本地运行：数据仅保存在本地，不上传任何服务器
- ✅ 高性能轻量：Rust 编译原生程序，老电脑流畅运行
